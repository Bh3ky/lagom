# Lagom Marketing OS (Next.js)

This is the Next.js App Router version of Lagom Marketing OS.

![Dashboard preview](public/dashboard.png)

## Tech Stack

- Next.js (App Router)
- React + TypeScript
- Tailwind CSS
- Gemini API (`@google/genai`)

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in `.env.local`
3. Run the app:
   `npm run dev`

## Production Build

1. Build:
   `npm run build`
2. Start:
   `npm run start`

## Tests

Run the test suite with:
`npm run test`
